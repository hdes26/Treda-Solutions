## Instalación 🔧

### _1. clone el repositorio_

```
git clone https://github.com/hdes26/Treda-Solutions.git
```

### _2. instale las dependencias_

```
npm install
```

### _3. inicialice el contenedor de docker_

```
docker-composer up

```
